Put project into Eclipse.
Build. 
Run.

You can choose local or network play.
Choose the same seed if you play Network in order to have the same map.
Host IP Address will be presented and Client will have to enter it.
Host uses Arrow keys to move and Space bar to place bombs.
Client uses WASD to move and Shift to place bombs.
